# code-assistant
 A Streamlit-powered AI Code Explainer &amp; Debug Assistant using Google Gemini 2.5 Flash. you can paste any code and in any language AI explains, finds bugs, suggests improvements, and gives time complexity.
Built with Streamlit + Google Gemini 2.5 Flash
This project is an AI-powered tool that analyzes code, explains it, detects bugs, suggests optimized versions, and estimates time complexity 


Tech Stack:


Python 3.10+

Streamlit

Google Generative AI (Gemini API)

dotenv (.env)

Pydantic

Installation:
1. Clone the Repository

git clone https://github.com/your-username/code-assistant.git
cd code-assistant

2. Install dependencies

pip install -r requirements.txt

3. Add your Gemini API key

GEMINI_API_KEY=AIzaSyXXXXXXXXXXXXXXX

4. Run the app

streamlit run streamlit_app.py
